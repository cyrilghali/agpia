Avva Shenouda — Coptic Unicode rendering demo

The Avva Shenouda font uses "CS encoding": Coptic glyphs are stored at ASCII code points (a → ⲁ, D → Ⲇ, 7 → ⲝ, …). The fix adds 65 cmap entries so the same glyphs are also reachable via their proper Unicode code points, while all legacy ASCII mappings are preserved.
